# Terms and Definitions
|Term|Definition|
|-|-|
|Blank|A Face with no pips (or the number 0).|
|Boneyard|where all Tiles start, are shuffled, and drawn from during play.|
|Depot|Central point for the Engine and all player Lines.  Mexican Trains may also start from the Depot.  Also called the Hub or Station.|
|Derail|Laying a Tile that ends in a number with no remaining unplayed Tiles.|
|Double|A Tile where both Faces have the same number of pips.|
|Engine|The initial Tile for the current Round.  All Lines start with a Face matching the Engine.|
|Face|Half of a Tile with a number of pips (or an actual number)|
|Game|All the Rounds of a particular playing session.|
|Hand|The Tiles held by a player in the current Round.|
|Line|A Sequence of Tiles laid end to end where the Faces of adjacent Tiles match.  All Lines start by matching Engine Face number.|
|Marked|A private Train that is temporarily Public allowing any player to lay Tiles on it.|
|Marker|A coin or token used to temporarily mark Private Trains as Public.|
|Private&nbsp;Train|A Line owned by a particular player who has exclusive access to lay Tiles against the Line.|
|Public&nbsp;Train|All Mexican Trains and all Marked Private trains, if any.|
|Round|The play from shuffling the Boneyard, drawing Hands, moving the Engine to the Depot, and laying Tiles until one player empties his Hand.|
|Sequence|A set of Tiles that match end to end and are played on a Line in a single turn.|
|Tile|A Domino with 2 numbered Faces|
